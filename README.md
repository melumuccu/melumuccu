# 主な成果物
## QryptoAle
### 概要
- 仮想通貨取引のサポート
  - 現在所有している各仮想通貨の「平均取得価額」や、「今売ると何円くらいの損益になるか」などを表示
- 取り扱う取引所
  - Binance
- CLIで動作
